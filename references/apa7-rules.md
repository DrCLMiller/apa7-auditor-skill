# APA 7 Audit Checklist — Full Rule Reference

## TITLE PAGE

| Check | Rule |
|-------|------|
| Paper title is **bold** | APA 7 §2.4 |
| Title is centered | APA 7 §2.4 |
| Author name is NOT bold | APA 7 §2.4 |
| Institutional affiliation present | APA 7 §2.6 |
| Course name and number present | APA 7 §2.7 |
| Instructor name present | APA 7 §2.7 |
| Assignment due date present | APA 7 §2.7 |
| Date format: Month Day, Year (no comma after month) | APA 7 general |
| No running head (student papers) | APA 7 §2.18 |
| Page number top-right, starting page 1 | APA 7 §2.18 |

| Level | Format | Rule |
|-------|--------|------|
| Level 1 | Centered, Bold, Title Case | §2.27 |
| Level 2 | Left-aligned, Bold, Title Case | §2.27 |
| Level 3 | Left-aligned, Bold Italic, Title Case | §2.27 |
| Level 4 | Indented, Bold, Title Case, ends with period | §2.27 |
| Level 5 | Indented, Bold Italic, Title Case, ends with period | §2.27 |

- Paper must begin with introductory content BEFORE first heading (§8.14)
- Don't label Introduction as a heading
- Headings must be hierarchical (no skipping levels)

## IN-TEXT CITATIONS

| Check | Rule |
|-------|------|
| Use author's **last name** only (never first name) | §8.17 |
| Use `&` inside parentheses; "and" outside | §8.17 |
| 1 author: `(Smith, 2020)` | §8.17 |
| 2 authors: `(Smith & Jones, 2020)` | §8.17 |
| 3+ authors: `(Smith et al., 2020)` from first citation | §8.17 |
| No citation for original author analysis | §8.1 |
| Every specific claim from a source must be cited | §8.1 |
| Page/para number for direct quotes | §8.23 |
| No comma between et al. and year | §8.17 |
| Institutional author: abbreviate if well-known after first use | §8.21 |
| If individual author identifiable, cite individual not institution | §8.21 |

## BLOCK QUOTES (Direct Quotations of 40+ Words)

| Check | Rule |
|-------|------|
| Block quote used for all direct quotations of 40 or more words | §8.27 |
| Block quote indented 0.5 inch from left margin | §8.27 |
| No quotation marks around the block quote | §8.27 |
| Block quote is double-spaced | §8.27 |
| Citation appears **after** the closing punctuation of the block quote | §8.27 |
| Citation includes page or paragraph number | §8.23 |
| Short quotes (under 40 words) use quotation marks inline; do NOT use block format | §8.26 |

## CITATIONS ↔ REFERENCES CROSS-CHECK

- Every in-text citation → must have a reference list entry
- Every reference list entry → must have at least one in-text citation
- Author names in citations must match reference list exactly (same spelling, same year)
- If citation year differs from reference year → flag

## REFERENCE TYPES — QUICK FORMAT GUIDE

### Journal Article
`Last, F. I. (Year). Title of article. *Journal Name*, *Volume*(issue), page–page. https://doi.org/xxxxx`

### Webpage / Online Article
`Last, F. I. (Year, Month Day). *Title of page*. Site Name. https://www.url.com`

### Book
`Last, F. I. (Year). *Title of book* (edition if applicable). Publisher. https://doi.org/xxxxx`

### Government Report / Legislative Document
`Organization. (Year). *Title of report*. Publisher. URL`

### Legal / Statute
`Name of Act, Volume Source § Section (Year). URL`
- Bill title is italicized
- Follow APA 7 §11.5 for U.S. federal and state statutes

### Congressional / Government Testimony
`Last, F. I. (Year, Month Day). *Title of testimony* [Testimony before Committee Name]. Organization. URL`

### DOI Format
- Always use URL format: `https://doi.org/xxxxx`
- Not italicized
- No period after DOI at end of reference (APA 7 removed that rule)
- If no DOI, use URL; if no URL, omit

## URLS & DOIs

| Check | Rule |
|-------|------|
| DOIs formatted as `https://doi.org/xxxxx` | §9.35 |
| DOI resolves (verify) | §9.35 |
| URLs resolve to actual source | §9.34 |
| No tracking parameters (e.g., `utm_source=chatgpt.com`) | §9.34 |
| Use production URLs, not staging/dev subdomains | §9.34 |
| No URL shorteners | §9.34 |
| No hyperlink formatting (no blue underline) — use plain text | §9.35 |

**When to flag a missing DOI/URL:**
- Flag as `UNVERIFIABLE — No DOI/URL` only for sources where a persistent link is reasonably expected: journal articles published after ~1995, webpages, online reports, government documents.
- Do NOT flag books, book chapters, or print-era works (pre-internet) for lacking a DOI/URL — these sources predate digital identifiers and their absence is normal and expected.
- Use judgment for older journal articles (pre-1990s) — a missing DOI is not unusual and should not be flagged as an integrity issue.
- For paywalled sources where the title, author, and year can be verified but full-text is inaccessible, use `UNVERIFIABLE — Paywalled` and note what was and wasn't confirmed.

## COMMON INTEGRITY FLAGS

| Flag | Description |
|------|-------------|
| `utm_source=chatgpt.com` in URL | Reference URL copied from ChatGPT output — AI-assisted reference generation |
| Dev/staging subdomain in URL | URL not publicly accessible |
| Institution listed instead of individual author | Misattribution when individual is identifiable |
| Wrong title | Title in reference list does not match the actual source title |
| Reference not verifiable | Possible hallucinated or ghost reference |
| Reference list entry with no in-text citation | Flag at 1+ — possible padding of reference list |
| In-text citation with no reference entry | Orphaned citation |
| `UNVERIFIABLE — Paywalled` | Source exists but full verification blocked by paywall; Trinity Check partially completed |
| `UNVERIFIABLE — No DOI/URL` | No persistent link available; applies only to digital-era sources where one is expected |
